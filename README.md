# Unit Converter App

A simple unit converter app built with **React Native**, **Expo**, and **NativeWind**.

## Features

- **Unit Conversion**: Seamlessly convert between:
  - Meters ↔ Feet
  - Celsius ↔ Fahrenheit
- **Live Updates**: Conversions happen instantly as you type.

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the app:
    ```bash
    npx expo start
    ```

## Expo Version

- **Expo**: ~54.0.25

## Dependencies

Key dependencies include:

- **React Native**: 0.81.5 - Framework for building native apps using React.
- **React**: 19.1.0 - Library for building user interfaces.
- **NativeWind**: ^4.2.1 - A utility-first styling library for React Native.
- **Expo Router**: ~6.0.15 - File-based routing for React Native and web.
- **React Navigation**: ^7.x - Routing and navigation for Expo and React Native apps.
